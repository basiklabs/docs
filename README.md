# Tracelit Docs

Documentation for [Tracelit](https://tracelit.io) — session replay, heatmaps, error tracking, real-time visitors, AI-powered alerts, and OpenTelemetry backend observability.

Published at [docs.tracelit.io](https://docs.tracelit.io).

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview changes locally:

```bash
npm i -g mint
mint dev
```

View your local preview at `http://localhost:3000`.
